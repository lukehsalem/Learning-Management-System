# README.md
Built a production-style LMS from scratch in C# using an agile sprint workflow on GitHub. 
The project evolved iteratively from a CLI console app through a .NET MAUI cross-platform 
GUI and culminated in an ASP.NET WebAPI backend — mirroring real-world software refactoring 
and architectural evolution.

Tech: C# · .NET MAUI · ASP.NET WebAPI · Entity Framework · GitHub Actions
Patterns: Singleton · Inheritance · MVC · REST
Workflow: 7 sprints · GitHub Issues · CI/CD via GitHub Actions
